# EU MAIOR - Cognitive Core

## Project Overview
This project is the foundational architecture for the EU MAIOR platform, focusing on High Performance and Human Development. Use the "The Mind View" dashboard to explore detailed psychometric profiles.

## Technologies
- **Frontend**: React, Vite, Tailwind CSS (Shadcn/UI), Recharts, React Flow.
- **Backend**: NestJS, Prisma, PostgreSQL (Supabase), Redis (BullMQ), LangGraph.js.
- **AI**: Google Gemini (Imagen), OpenAI (Text).

## Setup
1. Clone repository.
2. `npm install` in root (frontend) and `backend/` directories.
3. Configure `.env` in `backend/` (see `.env.example`).
4. Run backend: `cd backend && npm run start:dev`.
5. Run frontend: `npm run dev`.
